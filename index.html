romfs is a space-efficient, small, read-only filesystem for Linux and some Linux based projects. It is part of stock Linux kernels since about the version 2.1.25. All 2.4 and 2.5 kernels contain support for romfs, but depending on the distributor, it might not been compiled in.

Since romfs is read only, you can't simply copy files to it, you must use the program genromfs to generate an image, which then can be mounted.
